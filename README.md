# Flap Indicator

## Hardware

We use the following Board: https://os.mbed.com/platforms/ST-Nucleo-L432KC/


### Back
#### Pins
* LED
  * D10 -> L-1
  * D9 -> L-2
  * D6 -> L-3
  * D5 -> L-4
  * D4 -> L-5
  * D3 -> L-L 
* A0 -> AD (+ GND & 3V3)
* A7 -> UART_TX
* A3 -> Button

### Front
#### Pins
* LED
  * D10 -> L-1
  * D9 -> L-2
  * D6 -> L-3
  * D5 -> L-4
  * D4 -> L-5
  * D3 -> L-L 
* D0 -> UART_RX
* ?? -> LX-Vario/SF
